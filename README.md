# Проект "Dish"

## Описание проекта.

Данный проект, созданный на REST принципах, является частью будущего учебного проекта для изучения микросервисной архитектуры. 

## Стек.

- **Java 17**
- **Spring Boot 2.7.10**
- **Spring Data**
- **PostgreSQL 14**
- **lombok**


## Требования.

- **Java 17**
- **Maven 3.8**
- **PostgresSQL 14**
